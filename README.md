Варіант обирається за формулою V = (N mod K) + 1, де N - Ваш номер за академічним журналом групи, K=13, а mod - операція "залишок від ділення на".
V = (7 mod 13) + 1 = 8. Отже, виконуємо завдання варіанту 8.
